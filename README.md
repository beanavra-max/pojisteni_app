# PojištěníApp

Jednoduchá PHP aplikace pro správu pojištěnců.  
Projekt je postaven podle principu MVC (Model-View-Controller).

## Struktura projektu

- `index.php` – vstupní soubor
- `kontrolery/` – kontrolery
- `modely/` – modely
- `pohledy/` – view (HTML)
- `style.css` – stylování

## Jak spustit

1. Nainstalujte XAMPP nebo jiný lokální PHP server.  
2. Zkopírujte složku `pojisteni_app` do `htdocs` (nebo jiné kořenové složky serveru).  
3. Otevřete prohlížeč a zadejte:  http://localhost/pojisteni_app/

4. Vyplňte formulář a sledujte výsledky v tabulce.
